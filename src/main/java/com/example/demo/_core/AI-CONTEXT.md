# _core

## 목적
애플리케이션 전반에서 공통으로 사용되는 핵심 유틸리티 및 설정을 포함하는 패키지입니다.

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `.keep` | 패키지 구조 유지를 위한 빈 파일 |

## 하위 디렉토리
- `utils/` - 공통 유틸리티 클래스

## AI 작업 지침
- **Modern Java**: Java 21의 `var` 키워드를 적극적으로 사용하여 가독성을 높입니다.
- **주석**: 모든 로직에 친절하고 상세한 한글 주석을 작성합니다.
- **청결한 코드**: 사용하지 않는 임포트와 중복 코드를 즉시 제거합니다.

## 테스트
- 이 패키지의 개별 유틸리티에 대한 단위 테스트를 작성하여 검증합니다.

## 의존성
- 내부: 없음
- 외부: Spring Boot Starter
