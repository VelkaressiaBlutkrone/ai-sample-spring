# utils

## 목적
애플리케이션 내에서 공통으로 사용되는 범용 유틸리티 클래스들을 모아놓은 패키지입니다.

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `Resp.java` | API 응답 형식을 통일하기 위한 공통 응답 래퍼 클래스 |

## 하위 디렉토리
- 없음

## AI 작업 지침
- **Modern Java**: Java 21의 `var` 키워드를 적극적으로 사용하여 가독성을 높입니다.
- **제네릭 활용**: 유틸리티 클래스는 타입 안전성을 위해 제네릭을 적절히 활용합니다.
- **주석**: 모든 로직에 친절하고 상세한 한글 주석을 작성합니다.
- **청결한 코드**: 사용하지 않는 임포트와 중복 코드를 즉시 제거합니다.

## 테스트
- `Resp` 클래스의 응답 생성 로직을 검증하는 단위 테스트를 수행합니다.
