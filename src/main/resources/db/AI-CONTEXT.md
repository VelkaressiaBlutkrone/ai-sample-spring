# db

## 목적
데이터베이스 스키마 정의 및 초기 데이터 삽입을 위한 SQL 스크립트를 관리하는 디렉토리입니다.

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `data.sql` | 애플리케이션 시작 시 실행되는 초기 데이터 삽입 스크립트 (H2용) |

## 하위 디렉토리
- 없음

## AI 작업 지침
- **SQL 표준**: 가급적 표준 SQL을 사용하며, 특정 DB 전용 문법은 최소화합니다.
- **데이터 일관성**: 외래 키 관계를 고려하여 삽입 순서를 유지합니다.
- **한글 주석**: SQL 스크립트 내에서도 데이터의 용도를 한글로 설명합니다.

## 의존성
- 내부: Java 도메인 엔티티 구조(`User`, `Board`, `Reply`)와 일치해야 함
- 외부: H2 Database
